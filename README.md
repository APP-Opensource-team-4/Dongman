# Dongman
## 본 저장소는 모든 프로젝트 파일에 관한 총괄 저장소 입니다.

25.05.15 업데이트 
https://www.figma.com/design/tPXTx3xhPB6JhA1DWHtvTk/Untitled?node-id=0-1&t=jvUpc1puEwlt1ThD-1

#### 수정사항
- 앱 메인 컬러 설정 = 블루+화이트
- 로그인 회원가입창 수정
- 채팅창 화면 수정
- 지역 설정 창 추가
- 모집글 생성화면 수정
- 프로필 수정 창 수정

#### 추가 수정 필요
- 메인 화면 위에 알림창
- 모집글 작성 세부 내용 화면
- 지도/날씨앱 연동 화면 수정
- 동만이 메인 로고 수정(희망사항.. 화면 분위기 잡다보니 로고가 안 어울리는 것 같아서 조금 바꾸고 싶습니다..!)
